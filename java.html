Learn Java from Zero to Advanced

Powerful, Object-Oriented Programming Language

1. What is Java?

Java is a platform-independent, object-oriented programming language used for desktop apps, mobile apps (Android), web applications, and enterprise systems. It follows the principle “Write Once, Run Anywhere (WORA)”.

2. Basic Program Structure

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}

Explanation:

  • public class HelloWorld → Class declaration
  • public static void main → Main method, entry point of program
  • System.out.println() → Prints output

3. Variables and Data Types

Java is strongly typed. Every variable must have a type.

  • int → numbers
  • double → decimals
  • char → single characters
  • boolean → true/false
  • String → text
int age = 25;
double price = 19.99;
char grade = 'A';
boolean isStudent = true;
String name = "Ali";

4. Operators

  • Arithmetic: +, -, *, /, %
  • Comparison: ==, !=, <, >, <=, >=
  • Logical: &&, ||, !
int x = 10, y = 5;
System.out.println(x + y); // 15
System.out.println(x > y); // true
System.out.println(x == y); // false

5. Conditional Statements

int age = 18;
if (age >= 18) {
  System.out.println("Adult");
} else {
  System.out.println("Minor");
}

6. Loops

// For loop
for (int i = 0; i < 5; i++) {
  System.out.println(i);
}

// While loop
int j = 0;
while (j < 5) {
  System.out.println(j);
  j++;
}

7. Arrays

String[] fruits = {"Apple", "Banana", "Mango"};
for (String fruit : fruits) {
  System.out.println(fruit);
}

8. Methods

public class Calculator {
  static int add(int a, int b) {
    return a + b;
  }

  public static void main(String[] args) {
    System.out.println(add(5,3));
  }
}

9. Object-Oriented Programming (OOP)

Classes and Objects

class Car {
  String brand;
  int year;

  Car(String b, int y) {
    brand = b;
    year = y;
  }

  void display() {
    System.out.println(brand + " " + year);
  }
}

public class Main {
  public static void main(String[] args) {
    Car c1 = new Car("Toyota", 2022);
    c1.display();
  }
}

Inheritance

class Animal {
  void sound() {
    System.out.println("Animal makes sound");
  }
}

class Dog extends Animal {
  void sound() {
    System.out.println("Dog barks");
  }
}

public class Test {
  public static void main(String[] args) {
    Dog d = new Dog();
    d.sound();
  }
}

10. Exception Handling

try {
  int result = 10 / 0; // Error
} catch (Exception e) {
  System.out.println("Error: " + e.getMessage());
} finally {
  System.out.println("Always executes");
}

11. Collections Framework

import java.util.ArrayList;

public class ListExample {
  public static void main(String[] args) {
    ArrayList list = new ArrayList<>();
    list.add("Apple");
    list.add("Mango");

    for (String item : list) {
      System.out.println(item);
    }
  }
}

12. Advanced Features

  • Generics → ArrayList<String> list
  • Streams API
  • Lambda Expressions
  • Multithreading
list.stream()
    .filter(item -> item.startsWith("A"))
    .forEach(System.out::println);
