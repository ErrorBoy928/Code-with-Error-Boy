Learn Python from Zero to Advanced

Simple, Powerful and Beginner-Friendly Language

1. What is Python?

Python is a high-level, interpreted, and object-oriented programming language. It is famous for being easy to read and is widely used in web development, data science, AI, and automation.

2. Python Syntax

Python uses indentation instead of curly braces.

# Print a message
print("Hello, World!")

3. Variables

No need to declare types explicitly; Python detects them automatically.

age = 25
price = 19.99
name = "Ali"
is_student = True

4. Data Types

  • int → numbers
  • float → decimals
  • str → text
  • bool → True/False
  • list → [1,2,3]
  • tuple → (1,2,3)
  • dict → {"name":"Ali","age":21}
  • set → {1,2,3}
student = {"name": "Ali", "age": 21}
print(student["name"]) # Ali

5. Operators

  • Arithmetic: +, -, *, /, %
  • Comparison: ==, !=, <, >, <=, >=
  • Logical: and, or, not
x, y = 10, 5
print(x + y)  # 15
print(x > y)  # True

6. Conditional Statements

age = 18
if age >= 18:
    print("Adult")
else:
    print("Minor")

7. Loops

# For loop
for i in range(5):
    print(i)

# While loop
j = 0
while j < 5:
    print(j)
    j += 1

8. Functions

def greet(name):
    return "Hello " + name

print(greet("Ali"))

# Lambda function
add = lambda a, b: a + b
print(add(5, 3))

9. Collections

# List
fruits = ["apple", "banana", "mango"]
print(fruits[0])

# Tuple
numbers = (1, 2, 3)

# Dictionary
student = {"name": "Ali", "age": 21}
print(student["age"])

10. Object-Oriented Programming (OOP)

class Car:
    def __init__(self, brand, year):
        self.brand = brand
        self.year = year

    def display(self):
        print(self.brand, self.year)

c1 = Car("Toyota", 2022)
c1.display()

11. Exception Handling

try:
    result = 10 / 0
except Exception as e:
    print("Error:", e)
finally:
    print("Always executes")

12. Modules

Python allows code reuse with modules.

# math module
import math
print(math.sqrt(16))

13. Advanced Features

  • List comprehensions
  • Decorators
  • Generators
# List comprehension
squares = [x**2 for x in range(5)]

# Generator
def numbers():
    for i in range(3):
        yield i

for n in numbers():
    print(n)

# Decorator
def decorator(func):
    def wrapper():
        print("Before function")
        func()
        print("After function")
    return wrapper

@decorator
def hello():
    print("Hello!")

hello()
